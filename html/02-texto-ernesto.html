Mi primera experiencia creando un archivo HTML (Hyper Text Markup) desde cero

Fecha de la nota: 24 de octubre de 2025
Versión: v1.0beta

Hoy creé desde cero un archivo index.html en Visual Studio Code.

Proceso paso a paso

1. Abrí VS Code y creé una carpeta en el ordenador en la ruta htdoc/developerszgz y dentro de esta direccion se creo un archivo de texto que fue renombrado como index.html. Tambien aprendi de otra maneraen Visual Studio Code crear archivos llamando la carpeta desde la direccion creada.
2. Escribí la estructura básica escribiendo html y dandole tabulador. Esto me trae directamente la estructura basica de html.
3. En la seccion Body se escribi Hola Mundo.
Aqui se muestra un extracto del codigo: 

<!DOCTYPE html>
                <html lang="es">
                <head>
                  <meta charset="UTF-8">
                  <title>Hola Mundo</title>
                </head>
                <body>
                  <h1>¡Hola Mundo!</h1>
                  <p>Mi primera página web en VS Code.</p>
                </body>
                </html>

Errores y correcciones

El primer error que cometi es no habilitar las extensiones para visualizarlas. Ya que al crear un archivo de texto y renombr la extensiòn html se reflejaba el archivo como index.html.txt
Otro error que cometi es que no habia activado el servidor apache para luego corregir mi codigo en el servidor local.
Un fallo común fue olvidar cerrar una etiqueta <p>, lo que provocó que parte del contenido se mostrara fuera de lugar en el navegador. Esto me enseño a mantener una estructura limpia y validar con el inspector del navegador.

Atajos y comandos útiles aprendidos

Guardar: Ctrl + S
Saltos de linea en VS Code: ALt + Z
